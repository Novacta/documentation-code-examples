Welcome

SampleClassLibrary is a .NET library that provides support for operating on integers or on integer arrays.

The project targets all platforms implementing the .NET Standard, version 2.0.

Documentation

The current documentation includes the following topics.

  • SampleClassLibrary release notes

  • Operating on integers walks you through operations on integers.

  • Operating on integer arrays describes how to operate on arrays of integers.

  • Library reference contains reference information about library types by namespace.
