Welcome

Novacta.Documentation.CodeExamples is a .NET library that provides support for managing code examples in documentation files.

The project targets all platforms implementing the .NET Standard, version 2.0.

The library is licensed under the MIT license.

Documentation

The current documentation includes the following topics.

  • Novacta.Documentation.CodeExamples release notes

  • Tutorial includes topics showing how to define code examples in a console application, run them and generate text files to be inserted in a documentation.

  • Library reference contains reference information about library types by namespace.

  • SampleClassLibrary Namespaces points to a sample documentation obtained through the Novacta.Documentation.CodeExamples library.
